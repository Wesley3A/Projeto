Este site foi desenvolvido afim de atender os requisitos abaixo:

  1. Sistema = Nome do sistema (cada aluno terá seu nome no sistema com sistema_Aluno)

  2. Projeto = Nome do projeto para gerar as tarefas, onde o id_criador é o id do usuário que criou o projeto e id_sistema é o id do sistema vinculado ao projeto

  3. Prioridade = uma descrição de prioridade (Alta, Média, Baixa...).

  4. Usuario = nome de um usuário e id

  5. Projeto_usuario = vínculo dos usuários com o projeto, para em uma plataforma real permitir acesso somente a usuários vinculados

  6. Comentario = id_usuario criador do comentário, id_pai_comentario = o id do comentário anterior, como se fosse uma resposta

  7. Grupo = Grupo da tarefa, por exemplo, Grupo Comissões, tarefas relacionadas a comissões estariam relacionadas a este grupo, para posteriores relatórios

  8. Tarefa = Tarefas propriamente ditas, id_projeto vinculado, id_criado = usuário criador, id_dev = desenvolvedor a fazer, id_pai_tarefa se tiver uma tarefa anteriormente criada que gerou essa tarefa, por exemplo, colocar um campo no formulário, gerou uma tarefa de alteração da base de dados para comportar este novo campo, id_tipo_tarefa tipo da tarefa, is_status_tarefa = status da tarefa, tempo_realizado = tempo para concluir executado, id_priorida = prioridade da tarefa, complexidade = nível de complexidade da tarefa, impacto = ímpacto positivo no sistema, id_grupo

  9. Tarefa_status = status das tarefas

  10. Tarefa_tipo = tipo das tarefas (desenvolvimento, análise)
